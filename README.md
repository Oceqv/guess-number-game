## What This Project Does

* Gives the user **3 attempts** to guess a secret number
* Uses a `while` loop to control how many guesses are allowed
* Ends early if the correct number is guessed
* Prints a message when the user runs out of attempts

## Why I Made This

I built this project to:

* Practice **`while` loops**
* Learn how to control program flow using counters
* Get comfortable with **input, conditionals, and breaking loops**
* Strengthen Python fundamentals before moving on to more advanced security topics

## How It Works

* The game starts with 3 attempts
* Each loop represents one guess
* If the guess is wrong, attempts decrease
* If the guess is correct, the loop stops
